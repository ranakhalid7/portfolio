Tabarru' wa Irtaqi Platform

Web platform for clothing & furniture donation (منصة تبرع وارتق) · Madinah College of Technology (TVTC), KSA

This project addresses social and environmental challenges around discarded clothes and furniture by providing a unified web platform that links donors, charities, and beneficiaries. It improves how donations are listed, requested, filtered, and tracked—supporting fair redistribution, less waste, and clearer handover workflows.

Key Features:
  • Role-based flows for donor, charity, beneficiary, and admin
  • Donation CRUD with categories, status (e.g. available, distributed), and media
  • Search and filtering (category, status, keywords) with pagination
  • Request lifecycle: submit, approve, receive, and cancel where allowed
  • Admin dashboard for users, donations, charities, and reports
  • Secure sessions, CSRF checks, and PDO-backed MySQL access
Technologies & Tools:
PHP MySQL PDO Bootstrap HTML / CSS / JavaScript XAMPP phpMyAdmin Figma
Methodology:

Requirements → design → implementation → testing → maintenance following a Waterfall-style lifecycle, with manual test cases covering login, registration, donations, filtering, charity receive flows, and admin management.

AI mind

Web assistant with retrieval-augmented generation & optional web search

A conversational web app where the user talks to a language model that decides when to query a private knowledge base or the open web. The agent loop (model → tool calls → model) is implemented with LangGraph and LangChain tool bindings; execution can be traced with LangSmith when enabled.

Architecture & stack:
  • Frontend: Vue 3 + Vite (modern chat UI)
  • Backend: Node.js + Express
  • Agent: LangGraph cycle—LLM chooses tools, observes results, responds
  • Knowledge store: Pinecone index with integrated embeddings
  • Ingestion: script using upsertRecords to load content
  • Retrieval: searchRecords with a text query (current Pinecone path—not legacy LangChain vector flows)
Tools (agent):
  • search_knowledge_base — semantic search in Pinecone
  • search_web — web search via SerpAPI (or Google Custom Search when keys are configured)
Technologies:
Vue 3 Vite Node.js Express LangGraph LangChain Pinecone LangSmith SerpAPI
